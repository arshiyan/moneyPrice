# توضیحات
 نمایش قیمت ارز شامل دلار، یورو، درهم

## فراخوانی
برای استفاده از این کلاس میتوانید به این شکل عمل کنید
`require_once ('priceClass.php');
`
و برای نمایش قیمت 

`priceClass::sellUSD()
`
متد بالا قیمت فروش دلار را نمایش می دهد

### لیست متدها

- priceClass::sellUSD()
- قیمت فروش دلار

- priceClass::buyUSD()
- قیمت خرید دلار

- priceClass::sellEUR()
- قیمت فروش یورو

- priceClass::buyEUR()
- قیمت خرید یورو

- priceClass::sellAED()
- قیمت فروش درهم

- priceClass::buyAED()
- قیمت خرید درهم
